Detailed Description

HTML::parser HTML parser.

For testing purposes some functions are exposed publicly. Like DOCTYPE parsing. DOCTYPE consists of nested sections of the following possible types:

  1. , <? ... ?>, "...", '...'
  2. <![...]]>
  3. <!...> First group can not contain nested groups. Second group can contain nested groups of the same type. Third group can contain all other groups.

NOTE: Actualy HTML standard only includes the third group <!DOCTYPE ...>. So the first two groups parsing might be removed in the future.
